# Spotify Performance Analytics Dashboard

## Overview
This project analyzes Spotify Top 50 data to provide governance-focused insights into
song and artist performance. The dashboard enables KPI monitoring, trend analysis,
and drill-down reporting to support oversight, compliance, and decision-making.

## Business Objectives
- Monitor key KPIs (Total Songs, Distinct Artists, Avg Popularity, Duration)
- Compare Explicit vs Non-Explicit content performance
- Analyze distribution by album type and release year
- Identify top-performing artists and songs
- Enable drill-down analysis for governance and reporting use cases

## Tools & Technologies
- Power BI
- SQL
- Excel
- Python (data preparation & validation)

## Key Features
- KPI summary for executive-level reporting
- Explicit vs Non-Explicit content analysis
- Monthly and yearly trend analysis
- Artist- and song-level drill-downs
- Data validation and governance checks

## Dataset
- Spotify Top 50 dataset
- ~50K+ records after transformation
- Cleaned, validated, and standardized for reporting

## Governance & Controls
- Data validation and consistency checks
- Documented assumptions and KPI definitions
- Reproducible data transformations
